# 🐦 Flappy Bird

A simple Flappy Bird game made using **HTML**, **CSS**, and **JavaScript**.  
Fly the bird through the pipes and try not to crash!

---

## 📦 Features

- 🎮 Smooth game mechanics
- 🏆 Score tracking
- 💥 Collision detection
- 📱 Responsive layout
- 🔄 Restart game on crash
